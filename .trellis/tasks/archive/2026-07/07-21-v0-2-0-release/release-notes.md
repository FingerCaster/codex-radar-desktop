# Codex Radar Desktop v0.2.0

本次版本完善了桌面常驻体验：主界面设置入口、任务栏右键菜单与开机自启现已统一接入偏好设置；窗口位置支持持久化，并提供五个快捷位置预设。

- 全面更新主界面、详情页与设置页视觉设计。
- 为 Sol、Terra、Luna 使用本地太阳、地球、月亮模型图稿，其余模型使用 Codex 图标。
- 支持在主站雷达与分布式雷达之间切换，并保持来源隔离的缓存与刷新状态。
- 提供 Windows x64 的 MSI 与 NSIS 安装包。

注意：Windows 安装包未进行代码签名，系统可能显示安全提示。本版本不包含新的 macOS `0.2.0` 二进制文件。
